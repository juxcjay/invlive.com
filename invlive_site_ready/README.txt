INSTRUCTIONS:

1) Unzip invlive_site_ready.zip and you will get index.html and btc_qr.jpg.
2) Replace btc_qr.jpg with your real QR image if you want the real QR to display.
3) Upload both files to your GitHub repository 'invlive-site' using Add file -> Upload files.
4) Commit changes. Then enable Pages in Settings -> Pages -> Branch: main, Folder: /(root).
5) Wait 1-3 minutes and open: https://juxcjay.github.io/invlive-site/

Notes:
- If the page does not show updated currency values when opened from your file system, upload to GitHub Pages or run a local server:
  python -m http.server 8000
  then open http://localhost:8000
- If you want me to regenerate the ZIP with your exact QR image, upload the QR image here or tell me and I'll provide steps.
